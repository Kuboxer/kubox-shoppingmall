📋 API 엔드포인트

결제 검증:
POST /api/payment/verify

결제 취소:
POST /api/payment/cancel

결제 내역:
GET /api/payment/history

서비스 상태:
GET /api/payment/health

🆕 장애 모드 제어:
POST /api/payment/failure/toggle

🆕 장애 상태 확인:
GET /api/payment/failure/status

🧪 테스트

터미널에서 curl 명령어로 API 테스트:

curl -X POST http://localhost:8083/api/payment/verify -H "Content-Type: application/json" -H "User-Email: test@example.com" -d '{"order_id":"ORDER_001","receipt_id":"RECEIPT_001","price":25000,"order_name":"테스트 상품","buyer_name":"테스트고객","method":"card"}'
