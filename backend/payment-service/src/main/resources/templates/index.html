📋 API 엔드포인트

결제 준비:
POST /api/payments/prepare

결제 검증:
POST /api/payments/verify

사용자 결제내역:
GET /api/payments/user/{userId}

주문별 결제정보:
GET /api/payments/order/{orderId}

🧪 테스트

터미널에서 curl 명령어로 API 테스트:

curl -X POST http://localhost:8083/api/payments/prepare -H "Content-Type: application/json" -d '{"orderId":123,"userId":1,"amount":1000,"orderName":"테스트"}'

Port: 8083 | MySQL 연결됨 | BootPay 설정 완료
